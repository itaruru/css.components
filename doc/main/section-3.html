prefix is lt-

3.1 .lt-image

.lt-image は Smart Phone 向けのシンプルな横並び画像リスト。

<div class="lt-image {$modifiers}">
  <div class="inner">
    <a href="javascript:void(0);">
      <img src="../assets/icon.jpg" width="40" height="40" alt="alt">
    </a>
  </div>
</div>
