1.0 - Lists

スマートフォン用のいくつかのリストを定義

1.1 - .list-image

.list-image は Smart Phone 向けのシンプルな横並び画像リスト。

Default styling

 
<div class="list-image">
  <div class="inner">
    <a href="#">
      <img src="#" alt=" ">
    </a>
  </div>
</div>

1.2 - .list-simple

.list-simple は Smart Phone 向けのシンプルな縦並びリスト。

Default styling
タイトル
ラベル サブタイトル 
<div class="list-simple">
  <a href="#">
    <img src="#" alt=" ">
    <span class="m-tilte">タイトル</span>
    <br clear="all">
    <span class="m-act1">ラベル</span>
    <span class="m-act2">サブタイトル</span>
  </a>
</div>
